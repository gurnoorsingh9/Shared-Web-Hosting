# G-Host – Static Website Hosting Platform

## 📌 Overview

G-Host is a Flask-based static website hosting platform that enables users to upload, host, and manage static websites through a user-friendly web interface. The platform provides secure authentication, email verification, website management, visitor analytics, and an administrative dashboard for monitoring users and platform activity.

The project was developed using **Python (Flask)** and **MongoDB**, making it lightweight, scalable, and easy to deploy.

---

## 🚀 Features

### User Features

* User Registration & Login
* Email OTP Verification
* JWT-based Authentication
* Upload Static Website Files (HTML, CSS, JavaScript, Images)
* Website Management Dashboard
* File Manager
* Visitor Analytics
* Storage Usage Monitoring
* User Profile Management
